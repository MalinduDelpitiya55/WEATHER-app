## API Reference

#### Get all items

```http
  GET /api/items
```

| Parameter| Type    | Description                                |
| :--------| :-------| :------------------------------------------|

#### Get item

```http
  GET /api/items/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
